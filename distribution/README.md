# /distribution

Bumper sticker designs, print specifications, and distribution strategy.

## Scope

- Adversarial pattern bumper sticker designs optimized for LPR misclassification
- Print specifications: materials, dimensions, placement guidelines
- Distribution strategy for organic adoption within Flock-contracted municipalities
- Design iteration based on OCR testing results and field feedback
- Cost analysis for bulk production

## Design Requirements

- Must be visually acceptable as a normal bumper sticker to human observers
- Must be placeable in the plate-adjacent region captured by Flock cameras
- Must survive outdoor conditions: UV exposure, rain, temperature cycling, road debris
- Print materials must maintain adversarial properties — color accuracy under IR, retroreflective ink options where applicable
- Must induce confident misreads, not read failures — the distinction is the entire point
