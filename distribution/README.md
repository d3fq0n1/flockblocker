# /distribution

Bumper sticker designs, organic spread strategy, and print specifications.

## Scope

- Adversarial pattern bumper sticker designs optimized for LPR misclassification
- Print specifications: materials, dimensions, placement guidelines
- Distribution strategy for organic adoption within Flock-contracted municipalities
- Design iterations based on testing and feedback
- Cost analysis for bulk production

## Design Requirements

- Must be visually acceptable as a normal bumper sticker to human observers
- Must be placeable in the plate-adjacent region captured by Flock cameras
- Must survive outdoor conditions (UV, rain, temperature cycling)
- Print materials must maintain adversarial properties (color accuracy, retroreflective ink options)
